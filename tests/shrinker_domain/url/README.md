### Заметки

При создании доменного фильтра одно из требуемых полей - protocol. Это список, который может содержать элементы "http" и "https". Логика определения протокола в сообщении такова: если в протоколе из входного сообщения есть cisco_flow_ или huawei_ то там проставляется https, а в остальных случаях http.

Выходное сообщение для доменных фильтров по URLS отличается по набору полей от того, что в суженном потоке. Подробности можно посмотреть в файле testutils.py из текущей директории в функции transform_for_domain_filter_url (касается, как набора полей, так и нейминга некоторых из них).
